# LiteAce
整合 Ace Admin 及 Spring MVC 5 的系統架構  
JKD版本 : 1.8  
開發環境 : Eclipse  
Runtime Server : Apache Tomcat v9.0
  
專案匯入方式  
  左上角 File > Import > Git > Projects from Git(with smart import) > Clone URI
  
啟動伺服器後  
首頁網址:localhost:8080/LiteAce  
  
  帳號密碼隨意輸入即可登入

開發教學:localhost:8080/LiteAce/developer
